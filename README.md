# 北京优集客科技有限公司
##   公司介绍
     我们是一家全渠道的新零售公司，集线上线下的互联网公司;
     线上支持用户在APP端进行消费购买，线下我们有自己的实体店（2000-5000平米）进行消费;
     公司聚集了国内一流互联网公司和传统零售业的人才，现在天使轮获得数亿投资。
     发展方向好，新零售也是未来几年的热点.
